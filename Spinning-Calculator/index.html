Microsoft Edge Surf is a fun, built-in browser game available in Microsoft Edge that lets players ride the waves as a surfer, dodging obstacles like krakens, rocks, and other hazards. Inspired by classic endless runner games, Surf features smooth controls, multiple modes (including time trial and zig-zag challenges), and even supports keyboard, mouse, touch, and gamepad input. It's accessible offline by typing edge://surf in the Edge address bar, making it a great distraction when there's no internet connection. With colorful graphics and engaging gameplay, Microsoft Edge Surf adds a playful touch to the browser experience.

Support: The game supports computer and mobile
