2048 is a popular single-player puzzle gam where players slide numbered tiles on a 4x4 grid to combine them and create a tile with the number 2048. Each move slides all tiles in one direction, and tiles with the same number merge into one when they touch. The gam requires strategic thinking and planning to avoid filling up the grid, which ends the gam. Simple to learn but hard to master, 2048 has become a classic in web gams

Support: The game supports computer only
